◆ setTimeout()

virtual TftpServerOperationResult ITFTPServer::setTimeout ( const int  timeout)
pure virtual

Set timeout.

This is the time the server will wait for a client to send a request. If the client doesn't send a request within this time, the server will close the connection. The default is 300 seconds. If you want to disable the timeout, set the timeout to 0.

Parameters
[in]timeoutthe timeout in seconds.
Returns
TFTP_SERVER_OK if success.
TFTP_SERVER_ERROR otherwise.

Implemented in TFTPServer.

◆ stopListening()

virtual TftpServerOperationResult ITFTPServer::stopListening ( )
pure virtual

Request the TFTP Server to stop listening.

The success return of this function doesn't mean the server has stopped, it just means that the server will stop listening when it is ready. When the server stops listening, the startListening() function will return.

Returns
TFTP_SERVER_OK if success.
TFTP_SERVER_ERROR otherwise.
